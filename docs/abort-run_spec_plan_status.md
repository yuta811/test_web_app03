# ABORT RUN 仕様・実装計画・現状メモ

更新日: 2026-05-06

## 1. 仕様（確定）

- スマホの**横持ちプレイ前提**のブラウザゲーム。
- ただしゲーム画面は**縦方向固定**（ポートレート固定）とし、端末向きに合わせてゲーム世界を回転させない。
- 操作は画面タップのみ（長押し含む）。
- 地上:
  - 長押し中は加速
  - 離した瞬間にジャンプ
- 空中:
  - 非入力時は重力落下
  - 長押し中は縦速度を0へ近づける浮遊挙動
  - その代わり横速度が減速
- 核ミサイルロード進行が時間で増加。
- ロード進行はステージ上の追跡ライン（ワールド座標）とUI%表示で可視化。
- ゴール到達でクリア、ロード完了/追いつかれ/落下でゲームオーバー。

### 今回修正した点

- 端末向き依存の回転表示を廃止し、固定縦画面を中央レターボックス表示。
- 自キャラ初期Yを調整し、ジャンプ時に画面上で見切れにくいよう改善。
