# ABORT RUN 仕様・実装計画・現状メモ

更新日: 2026-05-06

## 1. 仕様（確定）

- スマホの**横持ちプレイ前提**のブラウザゲーム。
- ただしゲーム画面は**縦方向固定**（ポートレート固定）とし、端末向きに合わせてゲーム世界を回転させない。
- 操作は画面タップのみ（長押し含む）。
- 地上:
  - 長押し中は加速
  - 離した瞬間にジャンプ
- 空中:
  - 非入力時は重力落下
  - 長押し中は縦速度を0へ近づける浮遊挙動
  - その代わり横速度が減速
- 核ミサイルロード進行が時間で増加。
- ロード進行はステージ上の追跡ライン（ワールド座標）とUI%表示で可視化。
- ゴール到達でクリア、ロード完了/追いつかれ/落下でゲームオーバー。

3. プラットフォーム当たり判定
4. カメラ追従
5. ロード進行・ロードライン
6. ゴール/ゲームオーバー

### Phase 2（ゲーム性追加）
1. 減速床
2. ブースト床
3. 重力強化ゾーン
4. ロード加速ノイズ
5. 警告UI強化
6. クリアランク

### Phase 3（演出強化）
1. 背景演出・パーティクル
2. サウンド強化
3. タイトル/チュートリアル強化
4. ステージ拡張

## 3. 現状の実装状況

### 完了済み
- 単一 `index.html` でCanvasゲーム実装。
- 基本ゲーム状態（TITLE/READY/PLAYING/CLEAR/GAME_OVER/PAUSED）。
- 地上加速、離しジャンプ、空中浮遊、重力、空中減速。
- プラットフォーム着地判定。
- 減速床/ブースト床/重力ゾーン/ロード加速ノイズ/ゴール。
- ロード進行率、ロードライン表示、危険時の警告演出。

### 今回修正した点
- 表示レイアウトを**縦固定レンダリング**へ統一。
- 端末向き依存の回転表示を廃止し、固定縦画面を中央レターボックス表示。
- 自キャラ初期Yを調整し、ジャンプ時に画面上で見切れにくいよう改善。

### 未対応（今後）
- サウンド、演出、追加ギミック（レーザー/逆風/遅延アイテム等）の拡張。

## 4. 当初仕様との差分（追記すべき点）

以下は、初期の詳細仕様案にはあるが、現ドキュメントで記述が薄い/未反映のため追記候補。

1. **ロード遅延アイテム**
   - Phase 2 の要素として明文化（ロード率の直接減少 or 一時的なロード速度低下）。
2. **ジャンプパッド**
   - プレイヤー有利ギミックとして、ステージ設計上の位置づけを追加。
3. **逆風エリア**
   - 追加ギミック候補ではなく、Phase 2 以降の正式要素として扱うかを明記。
4. **ランク評価基準**
   - クリアランクの閾値（S/A/B/C/D）を暫定値でも仕様に固定。
5. **UI最小要件**
   - `LOAD%` / `DIST` の常時表示、警告演出の段階（注意/危険/限界）を定義。
6. **ステージ進行ガイド**

## 5. その他に追加すべき要素（実装前に決めると良い）

1. **判定の猶予値（トレランス）**
   - 着地判定・ロードライン接触判定の猶予を数値で定義し、理不尽な失敗を抑制。
2. **バランス調整用パラメータ表**
   - 速度・重力・浮遊係数・ロード速度倍率の初期値と調整レンジを一覧化。
3. **リザルトと再挑戦導線**
   - ゲームオーバー/クリア時の表示項目（タイム、ロード率、ランク）を固定。
4. **テスト観点の明文化**
   - 最低限のE2E観点（開始、ジャンプ、クリア、ゲームオーバー、リトライ）を仕様側に記載。
