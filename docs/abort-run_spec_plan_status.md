# ABORT RUN 仕様・実装計画・現状メモ

更新日: 2026-05-07

## 1. 仕様（確定）

- スマホの**横持ちプレイ前提**のブラウザゲーム。
- ただしゲーム画面は**縦方向固定**（ポートレート固定）とし、端末向きに合わせてゲーム世界を回転させない。
- 操作は画面タップのみ（長押し含む）。
- 地上:
  - 長押し中は加速
  - 離した瞬間にジャンプ
- 空中:
  - 非入力時は重力落下
  - 長押し中は縦速度を0へ近づける浮遊挙動
  - その代わり横速度が減速
- 核ミサイルロード進行が時間で増加。
- ロード進行はステージ上の追跡ライン（ワールド座標）とUI%表示で可視化。
- ゴール到達でクリア、ロード完了/追いつかれ/落下でゲームオーバー。

## 6. 2026-05-07 仕様確定（障害物リワーク）

### 6-1. 障害物の設計原則
- 見た目だけで効果が推測できることを必須要件にする。
- 「即死」「操作干渉」「進行干渉」「救済アイテム」を明確に分離する。
- 予兆（色・形・点滅）と接触後のフィードバック（画面フラッシュ）を必須化する。

### 6-2. 障害物仕様（確定）
1. **LASER GATE（即死）**
   - 見た目: 赤い縦ビーム + 点滅予兆。
   - 効果: 接触で即ゲームオーバー（`LASER CONTACT`）。
2. **MUD FLOOR（進行干渉）**
   - 見た目: 茶色の泥 + 波紋ライン。
   - 効果: 地上移動速度を強く減速。
3. **BOOST PAD（進行補助）**
   - 見た目: 水色シェブロン（>>）発光。
   - 効果: 横速度を一気に押し上げる。
4. **HEADWIND ZONE（操作干渉）**
   - 見た目: 紫の半透明エリア + 流線。
   - 効果: 重力強化により滞空を難化。
5. **JAMMER SPIKE（ロード加速トラップ）**
   - 見た目: 赤い警告三角 + 感嘆符。
   - 効果: 一定時間ロード進行速度を加速。
6. **DELAY CORE（救済）**
   - 見た目: 緑色の六角コア + 下向き矢印。
   - 効果: 取得時にロード進行率を直接減少。

### 6-3. ゲーム性を考慮した初期値（確定）
- LASER GATE 幅: 32px（可視性優先、狭すぎ禁止）
- MUD FLOOR 減速: `260`（停止はしない）
- BOOST PAD 最低到達速度: `610`
- HEADWIND ZONE 重力倍率: `1.58`（理不尽回避）
- JAMMER SPIKE 継続: `2.6s` / ロード倍率 `1.65`
- DELAY CORE 回復量: `-6.5%`

### 6-4. UI最小要件（確定）
- 常時 `LOAD%` / `DIST` / `SPD` を表示。
- 危険演出を3段階（注意/危険/限界）で色分け。
- 失敗原因を明示（`LASER CONTACT` / `MISSILE LAUNCHED` / `SIGNAL LOST` など）。
